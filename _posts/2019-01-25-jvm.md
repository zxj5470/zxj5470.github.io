# 集合
##  1. <a name='HashtableHashMapTreeMap'></a>Hashtable、HashMap、TreeMap有什么不同？
- Hashtable 是早期 Java 类库提供的一个哈希表实现。
    - 同步 (线程安全)
    - 不支持 null
- HashMap 
    - 非同步 (线程不安全)
    - 支持 null
- TreeMap
    - 基于红黑树顺序访问。
    - get，put，remove `O(log(n))`
